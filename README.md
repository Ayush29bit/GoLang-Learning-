# GoLang Learning

Learning Go by building small, practical projects.

This repository documents my journey of mastering Go through hands-on implementation instead of passive tutorials.

---

## Focus Areas

- Go fundamentals
- Structs and interfaces
- Error handling
- Concurrency (goroutines, channels, sync)
- net/http
- REST API development
- Backend architecture patterns

---

## Learning Strategy

- Study → Implement → Refactor → Push
- Build small projects continuously
- Prioritize clean, idiomatic Go
- Think in terms of production systems
